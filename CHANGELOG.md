# NeuroOffice Builder - История изменений

## v1.2 (Текущая версия)

### Добавлено:
- **Пресеты (наборы ролей)** - сохранение и загрузка разных наборов ролей и баз знаний
- **Статус LLM** - автоматическая проверка подключения каждые 10 секунд
- **Общая база знаний** - загрузка файлов с компьютера, список с чекбоксами вкл/выкл
- **Оркестрация** - отключенная по умолчанию (можно включить в настройках)

### Структура данных:
- LLMs: `backend/data/llms.json`
- Роли: `backend/data/roles.json`  
- Базы знаний: `backend/data/knowledge-bases.json`
- Пресеты: localStorage (браузер) - ключ `nob_presets`

### Как восстановить данные:

1. **Роли и LLM** - в браузере нажать "Синхронизация" на вкладке LLM

2. **Базы знаний** - загрузить заново через кнопку "Выбрать файлы с компьютера"

3. **Пресеты** - хранятся в браузере, для восстановления:
   - Открыть Chrome DevTools (F12)
   - Application → Local Storage → http://localhost:5173
   - Ключ `nob_presets` - содержит массив пресетов
