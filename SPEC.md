# NeuroOffice Builder — Спецификация

## 1. Product Overview

**Название:** NeuroOffice Builder  
**Тип:** Десктоп-приложение (Electron)  
**Назначение:** Конструктор нейроофисов — создание и управление AI-агентами с разными ролями  
**Целевая аудитория:** Малый и средний бизнес, предприниматели

---

## 2. Функциональность

### 2.1 LLM Подключения

| Тип | Параметры | Описание |
|-----|-----------|----------|
| OpenAI | API Key | GPT-4, GPT-4o, GPT-4o-mini |
| Anthropic | API Key | Claude 3.5 Sonnet, Haiku |
| Google | API Key | Gemini |
| Ollama | Endpoint | Локальные модели |
| LM Studio | Endpoint | Локальные модели |
| Custom API | URL + Key | Любой совместимый API |

### 2.2 Роли

| Поле | Тип | Обязательно |
|------|-----|-------------|
| Название роли | Строка | Да |
| Системный промпт | Текст | Да |
| LLM для роли | Выбор из подключённых | Да |
| Описание | Текст | Нет |

**Структура хранения ролей:**
```
/roles/
  ├── юрист/
  │   ├── role.json      # метаданные
  │   └── system.md     # системный промпт
  ├── маркетолог/
  │   ├── role.json
  │   └── system.md
  └── ...
```

### 2.3 Базы знаний

| Тип файла | Обработка |
|------------|-----------|
| .md | Конвертация в markdown |
| .json | Структурированные данные |
| .txt | Простой текст |
| .pdf | OCR через LiteParse |

**Структура хранения:**
```
/knowledge/
  ├── компания/
  │   ├── info.md
  │   └── прайс.json
  └── продукт/
      └── описание.md
```

### 2.4 Оркестратор

- Одна роль = "главный" (дирижёр)
- Дирижёр решает какую роль вызвать
- Задачи распределяются по ролям
- Результаты агрегируются

### 2.5 Интерфейсы запуска

| Интерфейс | Описание |
|-----------|---------|
| Telegram Bot | Для клиента |
| Web UI | Браузер |
| CLI | Для продвинутых |

## 4. Техническая реализация

### 4.1 Tech Stack

| Компонент | Технология |
|-----------|------------|
| Framework | Electron + React + TypeScript |
| UI | Tailwind CSS v4 |
| Стили | Radix UI |
| Иконки | Lucide React |
| LLM API | openai, anthropic, google-generativeai |
| Хранение | SQLite (better-sqlite3) |
| OCR | LiteParse (интегрируем) |

### 4.2 Структура проекта

```
neuro-office-builder/
├── src/
│   ├── main/           # Electron main process
│   ├── renderer/       # React UI
│   ├── core/           # Бизнес-логика
│   │   ├── llm/        # LLM адаптеры
│   │   ├── roles/      # Управление ролями
│   │   ├── knowledge/  # Базы знаний
│   │   └── orchestrator/ # Оркестратор
│   └── storage/        # SQLite
├── assets/             # Иконки, картинки
└── dist/              # Скомпилированное
```
